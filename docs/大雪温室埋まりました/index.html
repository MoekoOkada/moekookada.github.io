大雪。温室埋まりました。。。

2023.12.22 Moeko Okada

昨晩から新潟市内は珍しく大雪。
大学校内もすっかり雪化粧です。

膝下くらいまでは積もっています。
家から大学までのほんの短い距離を歩くのにもかなり時間がかかりました。
単純に雪道に慣れてないだけですが、滑らないように必死です。

流石に気になって温室を見に行ったら

見事に埋まってました。。。
中はビニールのおかげでやや暖かいので、植物は大丈夫そうです。

こういう光景は見慣れないので、まだ楽しめています。

おかだ
